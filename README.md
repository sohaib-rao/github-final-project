# Zakat Calculator

This repository contains a simple bash script that can be used to calculate Zakat, the obligatory Islamic charity. 

While the script uses standard financial variables (principal, rate, and time), it perfectly calculates your annual Zakat contribution. To use it for Zakat:
* **Principal:** Enter your total accumulated wealth (Nisab).
* **Rate of Interest:** Enter 2.5 (the standard Zakat percentage).
* **Time Period:** Enter 1 (for one lunar year).
